As a Product Designer with 6+ years of experience working on products serving 40M+ users (including Divar and IR-MCI), I've developed a keen eye for identifying UX opportunities that can enhance user engagement and learning outcomes. During my time using WordUp, I've identified several critical UX issues that, if addressed, could significantly improve user comprehension and retention.

Key Issues Identified:
  1. Unclear Learning Funnel Visualization – The relationship between "Uncovered Words," "Already Knew," "To Learn," and "In Progress" isn't visually obvious, making it difficult for users to understand their learning journey at a glance.
  2. Profile Image Misplacement – The profile image's placement next to statistics creates cognitive confusion about data ownership and personalization.
  3. Numbers Without Context – Several metrics lack clear labels and visual hierarchy, making it challenging for users to quickly understand their progress.
  4. Information Architecture – The hero section could benefit from a clearer information hierarchy to guide users toward their next learning action.
What I've Created:

I've invested time in creating a refined UI proposal that addresses these issues. The enhanced design includes:

  • Visual Learning Funnel – A step-by-step journey visualization that makes the learning process immediately clear
  • Improved Information Hierarchy – Separated profile section with personalized greeting and clear statistics
  • Enhanced Visual Design – Modern, clean interface with better color coding and iconography
  • Dark Mode Support – Full theme implementation for better accessibility
  • Responsive Design – Optimized for all device sizes

Note: This is a UI-only prototype (no backend connection) created to demonstrate the UX improvements visually.
